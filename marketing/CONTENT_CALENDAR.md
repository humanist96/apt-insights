## 콘텐츠 제작 가이드라인

### 블로그 글 구조
1. **서론**: 문제 제기, 독자 공감대 형성 (200자)
2. **본론**: 핵심 내용, 데이터, 사례 (1,500-2,500자)
3. **결론**: 요약, 행동 촉구 (200자)
4. **CTA**: 플랫폼 기능 연결

### SEO 체크리스트
- [ ] 제목에 핵심 키워드 포함
- [ ] 메타 설명 150자 이내
- [ ] H2, H3 태그 활용
- [ ] 이미지 alt 텍스트 작성
- [ ] 내부 링크 3개 이상
- [ ] 외부 권위 링크 1-2개

### 소셜 미디어 모범 사례
- **인스타그램**: 정사각형 1:1 비율, 밝은 컬러, 최대 30개 해시태그
- **트위터**: 280자 제한, 이미지/GIF 활용, 스레드로 긴 내용 분할
- **네이버 블로그**: 이미지 많이 삽입, 지역 키워드 강조

### 톤 & 보이스
- **전문적이지만 친근하게**: 복잡한 용어를 쉽게 설명
- **데이터 기반**: 주장에는 항상 수치와 근거 제시
- **행동 지향적**: 독자가 바로 실천할 수 있는 팁 제공
- **긍정적**: 투자 성공에 초점, 불안 조장 금지
